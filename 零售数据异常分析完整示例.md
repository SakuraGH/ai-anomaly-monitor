## 2. 操作步骤

### 2.1 上传数据并自动注册指标

```bash
curl -X POST http://localhost:8000/api/datasource/auto-import   -F "file=@零售示例数据.xlsx"   -F "metric_name=零售销售额"   -F "date_col=日期(按天)"   -F "value_col=销售额"   -F "dimension_cols=省份,城市,门店名称,商品名称,商品类别"
```

系统自动完成：
- 解析 Excel 文件，识别 13 列、40,514 行数据
- 自动推断日期列、数值列、维度列
- 注册"零售销售额"指标（优先级 P1）
- 立即触发监控管道

### 2.2 触发监控

```bash

curl -X POST "http://localhost:8000/api/monitor/run/零售销售额?target_date=2020-08-31"
```
